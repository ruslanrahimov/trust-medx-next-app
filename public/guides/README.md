# Гайды TrustMedX

Эта папка содержит PDF-файлы с руководствами для пациентов.

## Файлы

- `surgery-preparation.pdf` - Руководство по подготовке к операции
- `turkey-guide.pdf` - Всё о лечении в Турции
- `korea-guide.pdf` - Всё о лечении в Южной Корее
- `china-guide.pdf` - Всё о лечении в Китае

## Как заменить файлы-заглушки

1. Создайте или получите реальные PDF-файлы с содержанием
2. Замените пустые файлы в этой папке на реальные PDF
3. Убедитесь, что имена файлов соответствуют указанным выше
4. Файлы автоматически будут доступны для скачивания через секцию "Полезная информация" на странице для пациентов

## Структура интеграции

Пути к файлам указаны в словарях локализации:
- `/src/dictionaries/ru.json`
- `/src/dictionaries/en.json`
- `/src/dictionaries/ar.json`

Компонент: `/src/components/UsefulInfo.jsx`
